# XZ - Subway Surfers Style Game

A fun endless runner game inspired by Subway Surfers!

## Project Structure
- `src/` - Game source code
- `assets/` - Game assets (sprites, sounds, etc.)
- `docs/` - Documentation

## Getting Started

### Requirements
- Python 3.8+
- Pygame

### Installation
```bash
pip install pygame
```

### Running the Game
```bash
python src/main.py
```
